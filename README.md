# H - Offline Game Pack

This repository contains **10 local offline browser games**.

## How to use
1. Download or clone the repository.
2. Open `index.html` (or `./index.html`) in any modern browser.
3. Play from the organized categories.
4. Use each game's **Help** link for instructions.

## Included games
- Guess the Number (Puzzle)
- Memory Match (Puzzle)
- Minesweeper Lite (Puzzle)
- 2048 Lite (Puzzle)
- Tic-Tac-Toe (Strategy)
- Connect Four (Strategy)
- Rock Paper Scissors (Arcade)
- Snake (Arcade)
- Simon Says (Arcade)
- Hangman (Word)
